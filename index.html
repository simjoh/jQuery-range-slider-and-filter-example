jQuery UI Range Slider

CUSTOM EVENT (rangeSlide) with some filtering

A simple example using the jQuery rangeslide and some buttons to filter data. It is a quite naive implementation where the entire table is rerendered each time the slider handler is released and there is click on any of the buttons.

A more elegant way would be to check which the rooms that are added or deleted and just add these.

A more basic example can be found on the JQuery-UI manual page.
